node-hbase-thrift2
===

[![NPM Version][npm-image]][npm-url]
[![NPM Downloads][downloads-image]][downloads-url]

Yet another HBase thrift2 client.

# Installation

```sh
$ npm install node-hbase-thrift2
```

# Usage

```js
var HBase = require('node-hbase-thrift2')
```
